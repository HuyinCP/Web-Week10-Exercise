Key Features Demonstrated

This demo showcases various Spring Security 6 capabilities

Authentication

Form-based authentication with custom UserDetailsService and password encoding using BCrypt.

Authorization

Role-based access control with different user roles (USER, MODERATOR, ADMIN) and protected endpoints.

Data Persistence

Spring Data JPA with H2 in-memory database for user and role management with sample data.
